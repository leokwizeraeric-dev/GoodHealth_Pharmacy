# My Pharmacy Website

## About This Project

This is a simple pharmacy website I made while learning HTML and CSS. It is a demo project, not a real pharmacy website. I built this to practice making web pages.

## What I Used

- HTML
- CSS

No frameworks, no JavaScript, just plain code.

## What The Website Has

The website has these sections:

- **Navigation bar** - Links to jump to Home, Services, Products, Contact
- **Home section** - Welcome message about the pharmacy
- **Services section** - What the pharmacy offers (prescriptions, delivery, health advice)
- **Products section** - What they sell (pain medicine, vitamins, baby products)
- **Contact section** - Phone number, email address, location
- **Footer** - Copyright info

## How To Open

1. Download the two files: index.html and style.css
2. Put them in the same folder
3. Double click index.html
4. It opens in your browser

## Colors Used

- Green for the navbar and footer
- White for the main sections
- Gray for the boxes
- Dark green for headings

## What I Learned

- How to make a fixed navigation bar
- How to use hover effects on links and boxes
- How to add rounded corners with border-radius
- How to organize content with divs

## Is This A Real Pharmacy?

No. This is just a practice project. The pharmacy name, phone number, email, and address are all made up.
